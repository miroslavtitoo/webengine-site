# Webengine

Сайт студии цифровых решений. В референс взят [cappen.com](https://cappen.com/), повторяем логику/дизайн/анимации, наполнение — наше.

## Стек
Чистый HTML + CSS + JS, GSAP + ScrollTrigger для анимаций. Без сборки.

## Запуск локально
```
python -m http.server 8000
```
И открыть http://localhost:8000.

## Структура
```
index.html
css/      — reset.css, style.css
js/       — main.js
assets/
  img/    — плейсхолдеры
  video/  — место для showreel.mp4 (не коммитится)
  fonts/  — место для локальных шрифтов
```

## Что готово
- Прелоадер
- Hero со скролл-разворотом видео в full-screen и уходом вверх
- Statement (второй экран) с blur-in анимацией
- Сетка кейсов с появлением карточек
